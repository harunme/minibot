# docs/ 目录规则

本目录包含项目文档，按需参考：

- `ROADMAP.md` — V1-V4 版本路线图、里程碑和依赖关系
- `DECISIONS.md` — 项目技术决策历史（MQTT 选型、TTS 方案、存储策略等）
- `design/` — 各版本详细设计文档（按章节拆分到 `design/v1/` 等子目录）

## 文档引用规则

- 开发新模块前，阅读 `design/v1/` 下对应章节文件（而非完整 `V1_DESIGN.md`）
- 技术决策有疑问时，查阅 `DECISIONS.md` 了解决策背景
- 版本范围有疑问时，查阅 `ROADMAP.md` 确认边界

## 补充说明

- 路径特定的开发规则已拆分到 `.claude/rules/` 目录，AI 编辑对应路径文件时自动生效
- 本目录下的所有规则优先级低于根 `CLAUDE.md`
